# Vamshi Krishna - Personal Portfolio

Welcome to the repository for my personal portfolio website! This site showcases my projects, skills, certifications, and experience in software engineering and AI/ML.

## 🌐 Live Preview
You can view the live website here: **[Vamshi's Portfolio](https://VAMSHIKRISHNAKAMARI.github.io/portfolio/)**

## ✨ Features
- **Modern Design:** A sleek, cyberpunk-inspired dark theme with smooth animations.
- **Responsive Layout:** Works perfectly on desktops, tablets, and mobile phones.
- **Projects Showcase:** Detailed cards for my projects, including simulated interactive previews.
- **Contact Form:** A multi-step contact form that allows visitors to email me directly.
- **Resume Integration:** Easy access to download or view my latest resume.

## 🛠️ Tech Stack
- **HTML5:** For the core structure of the site.
- **Tailwind CSS:** For rapid and responsive styling.
- **JavaScript (Vanilla):** For interactive elements like modals and the multi-step contact form.

## 🚀 How to Run Locally
If you want to run this project on your own machine, it's very simple! No build steps or installations are required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VAMSHIKRISHNAKAMARI/portfolio.git
   ```
2. **Open the file:**
   Simply double-click the `index.html` file in the folder to open it in your web browser. That's it!
